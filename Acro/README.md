# SaaS Kits

Este repositório contém kits e recursos para desenvolvimento de aplicações SaaS.

## Estrutura do Projeto

- **Current/**: Versão atual do projeto
  - `schema.sql`: Schema do banco de dados
  - `database.types`: Definições de tipos do banco de dados
  - `context.md`: Documentação de contexto

- **Beta/**: Versões em desenvolvimento e experimentais

## Tecnologias

- SQL/PostgreSQL para banco de dados
- Documentação em Markdown

## Como Usar

1. Clone o repositório
2. Revise o schema do banco de dados em `Current/schema.sql`
3. Configure seu ambiente conforme necessário
